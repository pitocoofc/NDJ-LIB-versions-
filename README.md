# Ndj-lib | Lite Edition
This is the ultra-lightweight version of NDJ-LIB, optimized for maximum performance on mobile environments like Termux.
It focus on the core simplified syntax, removing non-essential modules to ensure a small footprint without sacrificing power.
MIT License included.
Pure Performance: Minimal MB usage.
Modular Ready: Compatible with the future ./dntsc module manager.

This version is not directly dedicated to developers due to a lack of role assignment and naming for maximum storage control.
